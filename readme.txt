Design a form visually with the block editor.  

Ideal for requests for quotations, order forms and more.

== Description ==
Design a form with the ease of the block editor and see the filled out forms in WordPress. Specially designed for more complex forms that remain responsive to the form factor of the device of the client or prospect.

== Installation ==
1. Install the plugin via Plugins -> New plugin. 
2. Activate the Light LMS Quizz plugin through the \'Plugins\' menu in WordPress
